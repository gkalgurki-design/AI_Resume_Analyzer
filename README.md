# AI Resume Analyzer

AI Resume Analyzer is a Machine Learning based web application that analyzes resumes and predicts suitable job roles based on skills and resume content.

---

## Features

- Upload Resume PDF
- Resume Text Extraction
- Job Role Prediction
- Skill Analysis
- Machine Learning Integration
- User Friendly Interface

---

## Technologies Used

- Python
- Flask
- Machine Learning
- Scikit-learn
- Pandas
- HTML
- CSS

## Installation

### 1. Clone Repository

```bash
git clone https://github.com/gkalgurki-design/AI_Resume_Analyzer.git
```

### 2. Open Project Folder

```bash
cd AI_Resume_Analyzer
```

### 3. Install Required Libraries

```bash
pip install -r requirements.txt
```

### 4. Run Application

```bash
python app.py
```
